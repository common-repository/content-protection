== Description ==
WordPress content protection plugin to protect content with login from logged out visitors.

**Core Features:**

* Protect All Content (Posts/Pages Everything)
* Protect Selected Content (Selected Post Type)
* Protect All Except Selected (All Except Selected Post Type)
* Set Custom Login Redirection URL

== Installation ==
1. Download the plugin zip file.
2. Extract the downloaded zip file.
3. Upload the plugin directory to the \"/wp-content/plugins/\" directory.
4. Activate the plugin through the \"Plugins\" menu in WordPress.
